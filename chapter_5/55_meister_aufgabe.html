Erstelle eine HTML Seite, die eine Registrierungsseite simulieren soll: Es gibt ein Username, Passwort und Passwort Wiederholen Feld. Bei der Eingabe soll dynamisch per Ajax auf die URL /exist-user (per PUT, im Body den User mit key 'user') eine Abfrage geschehen, die 200 zurückgibt, falls der Name noch frei ist, oder 409 wenn nicht. Das soll natürlich dem User visuell deutlich gemacht werden, dass der Username ok ist oder nicht. Die beiden Passwort Felder müssen natürlich auf Übereinstimmung geprüft werden.
