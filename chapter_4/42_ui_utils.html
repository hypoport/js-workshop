Einfache Animationen

Die Funktionen show() / hide() blenden Elemente ein/aus. Dabei wird das Layout neu berechnet, weil die CSS-Eigenschaft 'display:none' verwendet wird.
Die Funktionen fadeIn() / fadeOut() blenden Elemente ein/aus. Dabei wird das Layout neu berechnet, weil die CSS-Eigenschaft 'display:none' verwendet wird.
Die Funktionen toggle() blendet Elemente im Wechsel ein und aus. Dabei wird das Layout neu berechnet, weil die CSS-Eigenschaft 'display:none' verwendet wird. Über Parameter kann mein eine Animation hinzufügen.
